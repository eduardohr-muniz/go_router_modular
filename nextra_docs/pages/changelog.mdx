# 📝 Changelog

Histórico de mudanças e atualizações do GoRouter Modular.

## [2.0.0] - 2024-01-XX

### ✨ Novas Funcionalidades
- EventModule para comunicação entre módulos
- Sistema de chaves (Keys) melhorado
- Suporte completo a microfrontends
- Loader system aprimorado

### 🔧 Melhorias
- Performance otimizada para aplicações grandes
- Documentação completamente reescrita
- Exemplos práticos adicionados
- Testes unitários expandidos

### 🐛 Correções
- Correção de memory leaks no EventModule
- Estabilização do sistema de navegação
- Melhor tratamento de erros

### ⚠️ Breaking Changes
- Mudança na API de configuração inicial
- Reorganização da estrutura de módulos

## [1.5.0] - 2023-12-XX

### ✨ Novas Funcionalidades
- Shell modules para layouts compartilhados
- Navegação assíncrona
- Suporte a query parameters melhorado

### 🔧 Melhorias
- Injeção de dependências otimizada
- Melhor integração com GoRouter oficial

## [1.0.0] - 2023-10-XX

### 🎉 Lançamento Inicial
- Sistema modular de navegação
- Injeção de dependências
- Integração com GoRouter
- Documentação básica

---

Para atualizações mais detalhadas, confira os [releases no GitHub](https://github.com/Flutterando/go_router_modular/releases).
